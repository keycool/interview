# Story Interview Writer

通过深度访谈挖掘故事素材，融合参考风格，生成吸睛文章（公众号/X/微博）。

## 功能特点

- **三种文章类型**：深度长文（1500-2000字）、感悟短文（500-800字）、社交帖（100-200字）
- **精准身份定位**：两轮问答收集受访者背景，定制访谈问题
- **架构方案商讨**：根据素材推荐3个最适合的文章架构供选择
- **风格融合**：读取参考文章，融合风格特征
- **质量自检**：自动检查金句密度、节奏感、结构完整性

## 触发方式

```

开始访谈
```

## 访谈流程

```
1.1 确认文章类型（长文/短文/社交帖）
    ↓
1.2 受访者背景收集（两轮精准定位）
    ↓
1.3 访谈提问（根据身份定制问题）
    ↓
1.4 追问机制（挖掘细节）
    ↓
1.5 素材完整性检查
    ↓
1.6 架构方案商讨（3选1）
    ↓
1.7 写作确认
    ↓
2.x 自动写作（静默执行）
```

## 支持的文章类型

### 深度长文（1500-2000字）
- 适合公众号深度阅读
- 输出：3个备选标题 + 完整正文

### 感悟/简述类短文（500-800字）
- 适合公众号轻阅读
- 输出：3个备选标题 + 完整正文

### 社交帖（X/微博）（100-200字）
- 适合 X.com 或微博发布，精炼有力
- 输出：3个风格版本（情绪共鸣型、犀利观点型、故事叙述型）

## 支持的身份类型

- **创业者**：初创期/成长期/成熟期/二次创业/已退出
- **职场人**：新人/骨干/管理层/高管/专家路线
- **学生**：高中/本科/硕士/博士/刚毕业
- **自由职业者**：刚开始/稳定期/资深
- **转型期**：职业/行业/身份/地域/人生重启

## 安装方法

1. 将 `SKILL.md` 复制到 `~/.claude/skills/interview/` 目录
2. 在对话中输入触发词启动访谈
3. 按提示完成访谈流程
4. 选择架构方案后，AI 将自动生成完整文章

## 自定义风格

在 `风格参考/` 目录放入 `.md` 格式的参考文章，AI 会学习并融合这些风格。
