# Story Interview Writer

通过深度访谈挖掘故事素材，融合参考风格，生成吸睛文章（公众号/X/微博）。

## 功能特点

- **三种文章类型**：深度长文（1500-2000字）、感悟短文（500-800字）、社交帖（100-200字）
- **精准身份定位**：两轮问答收集受访者背景，定制访谈问题
- **架构方案商讨**：根据素材推荐3个最适合的文章架构供选择
- **风格融合**：读取参考文章，融合风格特征
- **质量自检**：自动检查金句密度、节奏感、结构完整性

## 触发方式

```
/interview
开始访谈
采访我
帮我写文章
写推文
发微博
写X帖子
```

## 访谈流程

```
1.1 确认文章类型（长文/短文/社交帖）
    ↓
1.2 受访者背景收集（两轮精准定位）
    ↓
1.3 访谈提问（根据身份定制问题）
    ↓
1.4 追问机制（挖掘细节）
    ↓
1.5 素材完整性检查
    ↓
1.6 架构方案商讨（3选1）
    ↓
1.7 写作确认
    ↓
2.x 自动写作（静默执行）
```

## 使用方法

1. 将 `SKILL.md` 复制到 `~/.claude/skills/interview/` 目录
2. 在对话中输入触发词启动访谈
3. 按提示完成访谈流程
4. 选择架构方案后，AI 将自动生成完整文章

## 自定义风格

在 `风格参考/` 目录放入 `.md` 格式的参考文章，AI 会学习并融合这些风格。
